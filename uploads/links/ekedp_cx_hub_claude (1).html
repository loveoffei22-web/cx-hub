☁️ OneDrive / SharePoint sync
Paste a direct-download JSON URL from OneDrive, SharePoint, Google Drive, Dropbox or a GitHub raw file. The ⟳ Sync button will fetch it.
How to get a direct link (3 quickest options) ▼
1. SharePoint (recommended for office)
Upload your snapshot.json to a SharePoint library → Share → "Anyone with the link". Copy the URL, then change ?e=... to ?download=1. Allows CORS.
2. OneDrive (personal)
Right-click snapshot.json → Share → "Anyone with the link, can view". Copy URL. Replace https://onedrive.live.com/...?resid=... with https://api.onedrive.com/v1.0/shares/u!<BASE64-URL>/root/content, OR simpler: use onedrive-direct.netlify.app to convert.
3. Dropbox / GitHub Gist (easiest)
Dropbox share link → change ?dl=0 to ?raw=1. GitHub: commit JSON to a public gist → use the "Raw" URL. Both allow CORS out of the box.
